# Prompt Debiasing

Ushbu sahifada promptlaringizni noxolislikdan xoli qilish uchun bir nechta oddiy usullar ko'rib chiqiladi.

## Namuna asosida tarafkashlikni kamaytirish

Ularning prompt ichidagi tarqalishi va tartibiga qarab, exemplars LLM natijalariga ta'sir ko'rsatishi mumkin (@si2022prompting). Bu masala ma'lum darajada [What's in a Prompt](https://learnprompting.org/docs/intermediate/whats_in_a_prompt) sahifasida muhokama qilingan.

### Tarqatish

Prompt ichidagi namunalar taqsimotini muhokama qilganda, biz nazarda tutayotgan narsa
qancha namunalar turli sinflardan mavjud. Masalan, agar siz
tvitterlarda binary  sentiment analysis  (ijobiy yoki salbiy) bajarish, va siz
3 ta ijobiy tvit va 1 ta salbiy tvit namunalarini taqdim eting, shunda sizda a
3:1 taqsimot. Taqsimot ijobiy tvitlar tomon og'ganligi sababli,
model ijobiy tvitlarni bashorat qilishga moyil bo'ladi.

### Buyurtma

Exemplarlar tartibi ham og'ishlarga sabab bo'lishi mumkin. Masalan, tasodifiy tartibda joylashtirilgan exemplarlar mavjud bo'lgan prompt
ko'pincha yuqoridagi promptdan yaxshiroq natija beradi, unda avval ijobiy tvitlar keltirilgan,
salbiy tvitlar bilan davom etdi.
