# Prompting Embeds-ni o'rganing


## Learn Prompting Embed nima?

ChatGPT sayti foydali, lekin agar siz promptlarni to'g'ridan-to'g'ri ushbu saytning o'zida yozib, sinab ko'ra olsangiz, yaxshi bo'lmasmidi? [Learn Prompting Embeds](https://embed.learnprompting.org/) yordamida, bu mumkin! Buni qanday sozlashni ko'rish uchun o'qishni davom eting. Biz ushbu interaktiv embedlarni ko'plab maqolalarda kiritamiz.

## OpenAI API kalitini oling

- Avval [platform.openai.com/account/api-keys](https://platform.openai.com/account/api-keys) sahifasiga o'ting
- So'ng, OpenAI akkauntingizga ro'yxatdan o'ting yoki tizimga kiring.
- **Create new secret key** tugmasini bosing. U matndan iborat satrni o'z ichiga olgan modal oynani ochadi:



- Ushbu kalitni nusxa ko'chirib, ushbu saytdagi embedga joylashtiring va **Submit** tugmasini bosing.

Endi siz ushbu sayt bo'ylab embedlardan foydalanishingiz mumkin bo'lishi kerak. E'tibor bering, OpenAI ushbu embedlar orqali yuborgan har bir prompt uchun sizdan to'lov oladi. Agar yaqinda yangi akkaunt yaratgan bo'lsangiz, 3 oy bepul kreditga ega bo'lasiz. Agar kreditlaringiz tugagan bo'lsa, xavotir olmang, chunki ushbu modellardan foydalanish juda arzon. ChatGPT har yetti ming so'z uchun atigi $0.02 turadi[^a].

## Embeddan foydalanish

## Xulosa

Ushbu embedlar sizga kurs davomida o'rganishni osonlashtiradi, chunki promptlaringizni tezda sinab ko'rishingiz mumkin, boshqa tabga o'tmasdan. Biroq, agar sizga ChatGPT interfeysi ko'proq yoqsa, embedlardan foydalanishingiz shart emas. Shunchaki promptlarni ChatGPT-ga nusxa ko'chirib joylashtirishda davom eting. Agar embedlardan foydalanmoqchi bo'lsangiz, API kalitingizni biror joyga yozib qo'ying, chunki OpenAI sayti uni faqat bir marta ko'rsatadi.

<Caution>
  API kalitingizni hech kimga aytmang, chunki ular sizning akkauntingizdan
  prompts.
</Caution>
