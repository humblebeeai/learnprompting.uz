Quyidagi misol rasmida (@ignore_previous_prompt) ko'rsatilganidek, hujumchi `user_input` ni o'zgartirib, prompt ni qaytarishga harakat qiladi. Mo'ljallangan maqsad goal hijacking (oddiy prompt injection) dan farq qiladi, bunda hujumchi `user_input` ni zararli ko'rsatmalarni chiqarish uchun o'zgartiradi (@ignore_previous_prompt).



Quyidagi rasm (@simon2022inject), yana `remoteli.io` misolidan olingan, quyidagilarni ko'rsatadi
Twitter foydalanuvchisi modeldan uning promptini oshkor qilishga erishmoqda.



Xo'sh, unda nima bo'pti? Nega kimdir prompt leaking haqida qayg'urishi kerak?

Ba'zan odamlar o'z promptlarini maxfiy saqlashni xohlashadi. Masalan, bir ta'lim kompaniyasi
men uchun buni 5 yoshli bola kabi tushuntirib bering degan promptdan foydalanayotgan bo'lishi mumkin
murakkab mavzular. Agar prompt sizib chiqsa, unda har kim undan foydalanishi mumkin bo'ladi
o'sha kompaniya orqali.
