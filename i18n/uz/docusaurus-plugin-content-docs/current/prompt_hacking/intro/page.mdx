# Kirish

</div>

Prompt hacking - bu LLMlarning zaifliklaridan foydalanadigan hujumlarni ifodalovchi atama bo'lib, ularning kirishlari yoki promptlarini manipulyatsiya qilish orqali amalga oshiriladi. An'anaviy xakerlikdan farqli o'laroq, u odatda dasturiy ta'minotdagi zaifliklardan foydalanadi, prompt hacking esa LLMni kutilmagan harakatlarni bajarishga majburlash uchun promptlarni ehtiyotkorlik bilan tuzishga asoslanadi.

Biz prompt hackingning uch turini ko'rib chiqamiz: prompt injection, prompt leaking va jailbreaking.
Har biri biroz boshqacha zaifliklar va hujum yo'llariga taalluqlidir, lekin barchasi LLM promptini manipulyatsiya qilish orqali kutilmagan natija olish tamoyiliga asoslanadi.

Prompt hackingdan himoyalanish uchun himoya choralarini ko'rish zarur. Bular promptga asoslangan himoya usullarini joriy etish, LLMning xatti-harakatlari va natijalarini muntazam kuzatib borish, shuningdek, fine tuning yoki boshqa texnikalardan foydalanishni o'z ichiga oladi. Umuman olganda, prompt hacking LLMlar xavfsizligi uchun tobora ortib borayotgan muammo bo'lib, bunday hujumlardan himoyalanish uchun doimo ogoh bo'lish va faol choralar ko'rish muhimdir.
