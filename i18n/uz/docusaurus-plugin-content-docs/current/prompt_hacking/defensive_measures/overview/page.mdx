# Umumiy ko'rinish

Prompt injectionning oldini olish juda qiyin bo'lishi mumkin va unga qarshi kuchli himoya usullari juda kam (@crothers2022machine)(@goodside2021gpt). Biroq, ba'zi oddiy yechimlar mavjud. Masalan, agar ilovangiz erkin matn chiqarishni talab qilmasa, bunday chiqishlarga ruxsat bermang. Promptni himoya qilishning ko'plab turli usullari mavjud. Bu yerda biz eng keng tarqalganlaridan ba'zilarini muhokama qilamiz.

Ushbu bobda so'zlarni filtrlash kabi qo'shimcha oddiy strategiyalar yoritiladi. Shuningdek, promptni yaxshilash strategiyalari (instruksiyani himoya qilish, post-prompting, foydalanuvchi kiritmalarini turli usullarda o'rash va XML teglash) ko'rib chiqiladi. Nihoyat, chiqishni baholash uchun LLMdan foydalanish va yana ba'zi modelga xos yondashuvlar muhokama qilinadi.
