# Filtrlash

Filtrlash - prompt hacking'ni oldini olish uchun keng qo'llaniladigan usuldir (@kang2023exploiting). Filtrlashning bir nechta turlari mavjud, lekin asosiy g'oya - boshlang'ich prompt yoki natijada bloklanishi kerak bo'lgan so'z va iboralarni tekshirishdir. Buning uchun blocklist yoki allowlist'dan foydalanishingiz mumkin (@selvi2022exploring). Blocklist - bu bloklanishi kerak bo'lgan so'z va iboralar ro'yxati, allowlist esa ruxsat berilishi kerak bo'lgan so'z va iboralar ro'yxatidir.
