# Xulosa

Prompt Injection hozirgi transformer arxitekturasi aslini ajrata olmasligidan kelib chiqadi
dasturchi ko'rsatmalari va foydalanuvchi kiritma ko'rsatmalari. Kelajakdagi modellarning buni ajrata olishi mumkinligi tasavvur qilinadi
bu ikki turdagi ko'rsatmalar, lekin hatto bu ham prompt injection ni to'xtatishga kafolat bermaydi. Hozirda, prompt injection bu
to'xtatish juda qiyin, va bu yaqin kelajakda ham muammo bo'lib qolishi ehtimoli katta.
