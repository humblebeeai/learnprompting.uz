# Kod injeksiyasi

Kod injeksiyasi (@kang2023exploiting) bu prompt hacking ekspluatatsiyasi bo'lib, unda hujumchi LLMni ixtiyoriy kodni (ko'pincha Python) bajarishga majbur qilishi mumkin. Bu holat asboblar bilan boyitilgan LLMlarda yuzaga kelishi mumkin, bunda LLM kodni interpreterga yuborishi mumkin, lekin bu LLMning o'zi kodni baholash uchun ishlatilganda ham sodir bo'lishi mumkin.

Kod injeksiyasi [amalga oshirilgani](https://twitter.com/ludwig_stumpp/status/1619701277419794435) haqida xabar berilgan, AI ilovasi [MathGPT](https://mathgpt.streamlit.app/)da va bu orqali uning OpenAI API kaliti olinib, foydalanilgan ([MITRE hisobot](https://atlas.mitre.org/studies/AML.CS0016/)).


*MathGPT shu vaqtdan beri kod injeksiyasiga qarshi himoyalangan. Iltimos, urinmang uni buzishga; ular API chaqiruvlari uchun to'laydi.*


## Misol

Keling, [MathGPT](https://mathgpt.streamlit.app/) ilovasining soddalashtirilgan misoli bilan ishlaymiz. Biz u matematik masalani qabul qilib, muammoni yechishga harakat qilish uchun Python kodi yozadi, deb faraz qilamiz.

Mana soddalashtirilgan misol ilovasi foydalanadigan prompt:

<AIInput>
Quyidagi matematik muammoni yechish uchun Python kodi yozing:
</AIInput>
