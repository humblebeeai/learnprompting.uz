# Ma'lumotlarni tuzish

LLMlar uchun oddiy va qiziqarli foydalanish holatlaridan biri bu ma'lumotlarni jadvallarga tartibga solishdir. Ehtimol, sizda bir nechta yangiliklar maqolalari yoki biznes hisobotlari bor va siz barcha muhim nuqtalarni jadvalga jamlashni xohlaysiz, shundan so'ng uni jadval fayliga yoki ma'lumotlar bazasiga joylashtirishingiz mumkin. Chat botlar, masalan, ChatGPT, sizga bu ishda yordam bera oladi.

Quyidagi hisobotdan ma'lumotlarni olish uchun unga `Generate a table containing this information:` so'zini qo'shishimiz mumkin.
