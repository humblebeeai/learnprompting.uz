## Kod generatsiyasi

ChatGPT yoki boshqa har qanday LLM asosidagi AI chat-botdan kod generatsiya qilishni so'rash juda oddiy:

```text
Junior python dasturchisi sifatida harakat qiling. Iltimos, har birida 6 tadan element bo'lgan 3 ta massivdan: predloglar, fe'llar va otlardan madlibs generatsiya qiluvchi kod yozing, otlar to'g'ridan-to'g'ri ob'ekt sifatida ham ishlatiladi. Kodni yozgandan so'ng uni bajaring.
```

Natija quyidagicha bo'ladi:

```python {1-13}
import random

prepositions = ['at', 'in', 'on', 'with', 'under', 'over']
verbs = ['run', 'jump', 'dance', 'sing', 'laugh', 'cry']
nouns = ['cat', 'dog', 'tree', 'flower', 'book', 'computer']

preposition = random.choice(prepositions)
verb = random.choice(verbs)
noun = random.choice(nouns)

madlib = f"I {verb} {preposition} the {noun}."

print(madlib)
```

## Kodni izohlash va formatlash

Botdan kodga izohlar qo'shishni va uni o'qish uchun qulayroq formatlashni so'rashingiz mumkin. Prompt ko'rsatmasidan so'ng, kodni tozalash uchun promptingizdan keyin uchta hash belgisi qo'shing:

```python
Please add line comments to this Python code and reformat it for legibility:
###
import math
amplitude = 30
frequency = 2
num_points = 50
for i in range(num_points):
    y = int(round(amplitude * math.sin(2 * math.pi * frequency * i / num_points)))
    print(' ' * (amplitude + y) + '*')
```

Natija quyidagicha bo'ladi:

```python {1-15}
# Import the math module
import math

# Set the amplitude, frequency, and number of points in the wave
amplitude = 30
frequency = 2
num_points = 50

# Loop over each point in the wave
for i in range(num_points):
    # Calculate the y value of the wave using the sine function from the math module
    y = int(round(amplitude * math.sin(2 * math.pi * frequency * i / num_points)))

    # Print a space for padding, followed by an asterisk at the appropriate position on the wave
    print(' ' * (amplitude + y) + '*')
```

## Nosozliklarni tuzatish

LLM asosidagi AI chat-botlar koddagi sintaksis xatolarini aniqlash bilan birga, kod bajarilganda yuzaga keladigan mantiqiy xatolarni ham topa oladi. Quyida Python skriptiga misol keltirilgan bo'lib, unda 3-qator mantiqiy xato tufayli 4-qatorida nolga bo'lish xatosi yuzaga keladi. Xatoni topish va tuzatish uchun ushbu oddiy promptni sinab ko'ring:

```python
Please debug this Python code:

1. pizza_cost = 100
2. partygoers = 10
3. while partygoers = 0:
4.    cost_per_person = pizza_cost / partygoers
5.    print("the cost per person is $", "{:.2f}".format(cost_per_person))
6.    partygoers -= 2
```

Natijada to'g'rilangan while sikli bo'ladi, u bo'luvchi o'zgaruvchisi musbat bo'lib qolishini ta'minlash uchun o'zgartirilgan:

```text {1-2}
3. while partygoers >
 0:
```

Yana murakkabroq xatolar uchun sizga ko'proq tafsilotlar kerak bo'lishi mumkin: xato xabari, muhit o'zgaruvchilari, ulashgan kutubxonalar yoki kiruvchi ma'lumotlardagi farqlar.

## Kod optimallashtirish

Agar skriptingizni bajarish juda ko'p vaqt olayotgan bo'lsa, ChatGPT kabi AI chat-bot uni optimallashtirishi mumkin. Mana, tuban samaradorlikka ega bo'lgan tub sonlarni topuvchi skriptni optimallashtirish misoli:

```python
Act like a very senior python developer. Please optimize this script:

for num in range(1, 101):
    if num  1:
        for i in range(2, num):
            if (num % i) == 0:
                break
        else:
            print(num)
```

Natijada tub sonlarni topish uchun [Eratosthenes elagi algoritmi](https://www.geeksforgeeks.org/sieve-of-eratosthenes/)dan foydalanadigan skript bo'lishi mumkin:

```python {1-6}
numbers = list(range(2, 101))
for i in numbers:
    for j in range(i * 2, 101, i):
        if j in numbers:
            numbers.remove(j)
print(numbers)
```

E'tibor bering, biz botdan skriptni optimallashtirish uchun "katta tajribaga ega dasturchi sifatida harakat qiling" deb so'radik. Shuningdek, siz undan ma'lum bir sohada tajribaga ega bo'lishini (masalan, saralash algoritmlari) yoki necha yillik tajribaga ega bo'lishini so'rashingiz mumkin. Yoki, agar sizda haddan tashqari murakkab ko'ringan skript bo'lsa, undan bu skriptni "juda yangi boshlovchi dasturchi sifatida" yozishni so'rashingiz mumkin.

## Dasturlash tillari o'rtasida tarjima qilish

AI chat-botlarining juda foydali funksiyalaridan biri â€” dasturlarni bir tildan boshqasiga o'tkazish imkoniyatidir. Bunga odatiy misol sifatida eski tildagi kodni zamonaviy tilga yangilash keltirilishi mumkin:

```COBOL
Act like an experienced developer with knowledge of both COBOL and Python. Please convert this COBOL program to Python:


IDENTIFICATION DIVISION.
PROGRAM-ID. CALCULATE-SS-TAX.

DATA DIVISION.
WORKING-STORAGE SECTION.
01 SALARY PIC 9(7)V99.
01 SS-TAX PIC 9(7)V99.

PROCEDURE DIVISION.
DISPLAY "Enter employee's salary: ".
ACCEPT SALARY.

COMPUTE SS-TAX = SALARY * 0.062.

DISPLAY "Social Security tax is: $", SS-TAX.
STOP RUN.
```

Natija quyidagicha bo'lishi mumkin:

```python {1-4}
salary = float(input("Enter employee's salary: "))
ss_tax = salary * 0.062

print("Social Security tax is: $", round(ss_tax, 2))
```

## Bir nechta fayllar bilan ishlash va kod bazasini refaktor qilish

Agar siz AI chat-botdan **bir nechta** fayllarni baholashni xohlasangiz, shunchaki fayl nomlarini ko'rsating va har birining kodini joylang:

```python {15-16}
My Python script uses two files, listed below, in different directories. Please display the output from executing conversation.py:

BEGIN FILE 1: ./phrases/coming_and_going.py
def hi(name):
   print(f"Hello, {name}!")
def bye(name):
   print(f"Goodbye, {name}!")

BEGIN FILE 2: ./conversation.py
from phrases.coming_and_going import hi, bye
hi("John")
bye("John")


Hello, John!
Goodbye, John!
```

Ushbu usul yordamida siz bir nechta fayllarni bitta promptda botga yuborib, kodni refaktor qilishni so'rashingiz mumkin. Yoki, har birini "yodlab olishini" so'rab, keyingi promptda uni bajarishni so'rashingiz mumkin.

## Ma'lumotlar bazasi serverini simulyatsiya qilish

Ko'pchilik AI chat-botlar kod bilan ishlaydi, lekin ba'zilari ma'lumotlar bazasi, veb va email serverlari kabi server muhitlarini ham simulyatsiya qila oladi. Mana bunday promptga misol:

```text
Microsoft SQL Server sifatida harakat qiling. "politics" nomli ma'lumotlar bazasi yarating va uning ichida "politicians" nomli jadval yarating. Uni 1900-2000 yillar oralig'idagi turli davrlardan dunyodagi mashhur siyosatchilarning 50 ta qatori bilan to'ldiring. To'liq ismlari, mamlakati, tug'ilgan sanasi va agar mavjud bo'lsa, vafot etgan sanasi uchun ustunlar qo'shing. Eng uzoq yashagan 3 siyosatchi uchun view yarating. Ushbu view tarkibini chiqaruvchi Transact-SQL buyruqini yarating va bajaring.
```

Natija quyidagicha bo'ladi:

```SQL {1-17}
-- Create the "politics" database
CREATE DATABASE politics;
GO

-- Use the "politics" database
USE politics;
GO

-- Create the "politicians" table
CREATE TABLE politicians (
    id INT PRIMARY KEY,
    full_name VARCHAR(50),
    country VARCHAR(50),
    date_of_birth DATE,
    date_of_death DATE
);
-- etc.
```

Shundan so'ng, ustunlar, indekslar va boshqalarni qo'shishingiz mumkin:

```
Add a column to the politicians table for the second-in-command and update it for each row.
```

Natija quyidagicha bo'lishi mumkin:

```SQL {1-11}
ALTER TABLE politicians ADD second_in_command VARCHAR(255);

UPDATE politicians
SET second_in_command =
    CASE name
        WHEN 'Winston Churchill' THEN 'Clement Attlee'
        WHEN 'Franklin D. Roosevelt' THEN 'Harry S. Truman'
        WHEN 'Mao Zedong' THEN 'Lin Biao'
        -- add more cases for each politician
        ELSE NULL
    END;
```

```
Act as Debian Linux command shell. Please respond to my commands as the terminal would, with as little explanation as possible. My first command is: ls -l
```

