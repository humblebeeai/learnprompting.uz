# Kirish

Endi siz ba'zi juda oddiy prompt engineering usullarini o'rganganingizdan so'ng, oddiy, kundalik muammolarni hal qilish uchun qanday foydalanishni o'rganish vaqti keldi. Ushbu bo'limda elektron pochta yozish/yig'ish va shartnomalar tuzish kabi vazifalar yoritiladi.

Quyidagi maqolalarda keltirilgan misollar sizga prompt engineeringni qo'llash mumkin bo'lgan sohalar doirasini ko'rsatishga yordam beradi.


*Ushbu misollarda biz ham ChatGPT, ham GPT-3 dan foydalanamiz. Odatda bu ahamiyatli emas qaysi biridan foydalansangiz.*
