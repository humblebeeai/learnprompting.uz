# Takrorlash

Prompt ichida bir xil so'zni yoki o'xshash iboralarni takrorlash modelni
yaratilgan tasvirdagi o'sha so'zga urg'u berishga olib kelishi mumkin (@oppenlaender2022taxonomy). Masalan, [@Phillip Isola](https://twitter.com/phillip_isola/status/1532189632217112577) DALLE yordamida quyidagi sharsharalarni yaratgan:

`A beautiful painting of a mountain next to a waterfall.`



`A very very very very very very very very very very very very very very very very very very very very very very beautiful painting of a mountain next to a waterfall.`



`very` so'ziga urg'u berilishi generatsiya sifatini yaxshilayotganga o'xshaydi! Takrorlash
mavzu so'zlariga ham urg'u berish uchun ishlatilishi mumkin. Masalan, agar siz
begona mavjudotlar bilan sayyora tasvirini yaratmoqchi bo'lsangiz, `A planet with aliens aliens aliens aliens aliens aliens aliens aliens aliens aliens aliens aliens` promptidan foydalanish
natijaviy tasvirdagi begona mavjudotlar ehtimolini oshiradi. Quyidagi tasvirlar Stable Diffusion yordamida yaratilgan.

`A planet with aliens`



`A planet with aliens aliens aliens aliens aliens aliens aliens aliens aliens aliens aliens aliens`
