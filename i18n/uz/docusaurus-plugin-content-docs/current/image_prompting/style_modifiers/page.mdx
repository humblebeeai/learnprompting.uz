# Uslub modifikatorlari

Uslub modifikatorlari oddiygina tavsiflovchilardir, ular doimiy ravishda
muayyan uslublarni hosil qiladi (masalan, 'qizil rangga bo'yalgan', 'shishadan yasalgan', 'Unityda render qilingan')(@oppenlaender2022taxonomy). Ularni birga birlashtirish mumkin
yanada aniqroq uslublarni hosil qilish uchun. Ular "san'at davrlari, maktablari va uslublari, shuningdek, san'at materiallari va ommaviy axborot vositalari, texnikalar va rassomlar haqidagi ma'lumotlarni o'z ichiga olishi mumkin"(@oppenlaender2022taxonomy).

## Eslatmalar

Oppenlaender va boshqalar(@oppenlaender2022taxonomy) `rendered in ...` tavsiflovchini
sifat kuchaytiruvchi deb ta'riflaydi, lekin bizning ishchi ta'rifimiz boshqacha, chunki bu modifikator doimiy ravishda aniq Unity (yoki boshqa render dvijok) uslubini hosil qiladi. Shuning uchun, biz bu tavsiflovchini uslub modifikatori deb ataymiz.
