# Sifat kuchaytiruvchilari

Sifat kuchaytiruvchilari(@oppenlaender2022taxonomy) bu yaratilgan tasvirning
ba'zi uslubga xos bo'lmagan sifatlarini yaxshilash uchun promptga qo'shiladigan atamalardir. Masalan, "ajoyib", "go'zal" va "yaxshi sifat" barchasi sifat kuchaytiruvchilari bo'lib, yaratilgan tasvir sifatini oshirish uchun ishlatilishi mumkin.


## Eslatmalar

Oldingi sahifadagi eslatmaga o'xshash tarzda, bizning sifat kuchaytiruvchilari bo'yicha ishchi ta'rifimiz Oppenlaender va boshqalar(@oppenlaender2022taxonomy)dan farq qiladi. Shuni aytish mumkinki,
ba'zan sifat kuchaytiruvchilari va uslub modifikatorlarini aniq ajratish qiyin bo'lishi mumkin.
