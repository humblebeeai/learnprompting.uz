# PromptChainer

[PromptChainer](https://arxiv.org/pdf/2203.06566.pdf)(@wu2022promptchainer) prompt muhandisligi uchun tadqiqot vositasidir. U promptlarni bir-biriga ulashga e'tibor qaratadi va eng
mukammal tadqiqot IDEsi deb topganman.

## Xususiyatlari

PromptChainer zanjir ko‘rinishiga, shuningdek, alohida tugun ko‘rinishiga ega. Ular, shuningdek,
nosozliklarni tuzatish funksiyalarini taqdim etadi.
