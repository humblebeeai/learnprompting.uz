# Dream Studio

[Dream Studio](https://beta.dreamstudio.ai/dream) bu Stable Diffusion yordamida tasvirlar yaratish uchun mo‘ljallangan veb-sayt,
matndan tasvirga modelidir.



## Xususiyatlari

Dream studio’ning asosiy xususiyati turli konfiguratsiya parametrlarini sozlash imkoniyatidir
modelning, masalan, yaratilgan tasvirning balandligi/eni va steps/sampling algoritmi. Bu in/outpainting imkonini beradi.

Hozirda u to‘liq ishlab chiqilmagan tarix ko‘rish funksiyasiga ega (u tasvirlarning piksellangan versiyalarini brauzer keshiga saqlaydi, lekin ularni ko‘rish imkoni yo‘q).
