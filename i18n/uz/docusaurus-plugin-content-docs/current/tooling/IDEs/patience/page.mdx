# Sabr

[Patience](https://www.patience.ai) bu turli xil tasvirlarni yaratish uchun mo'ljallangan veb-sayt
matndan tasvirga modellaridan foydalanadi.



## Xususiyatlar

Patience papka yaratish, kolleksiyalar/tarixni saqlash va bir nechta turli modellarni tanlash imkoniyatiga ega.
Shuningdek, u turli diffuziya sozlamalarini, masalan, o'lcham va namunaviy usulni o'zgartirishga imkon beradi.
