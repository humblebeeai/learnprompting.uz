# The Forge AI

[The Forge AI](https://theforgeai.com/) kuchli prompt IDE bo'lib, ilovalar yaratish va monetizatsiya qilish imkonini beradi. Siz turli xil sozlanadigan kirish turlaridan foydalanishingiz, shartli oqimlarni belgilashingiz, turli modellardan foydalangan holda matn va tasvirlar yaratishingiz mumkin. (Yana ko'plab funksiyalar qo'shiladi).



## Xususiyatlar

Ilova tayyor bo'lgach, uni xohlagancha shaxsiy yoki ommaviy tarzda boshqalar bilan ulashishingiz mumkin.
Ilovalarni marketplace'da token sotish yoki obuna orqali ham monetizatsiya qilish mumkin.
