# Human Loop

[Human loop](https://humanloop.com/) GPT-3 Playground’ga o‘xshash maydonchani taklif qilayotganga o‘xshaydi,
lekin ba’zi qo‘shimcha funksiyalar bilan. Hozirda ular ko‘proq sanoat foydalanuvchilari bilan ishlamoqda.



## Xususiyatlar

1. GPT-3 so‘rovlari va foydalanuvchi fikr-mulohazalarini log qilish uchun oddiy SDK taqdim etadi.
2. Foydalanuvchi kiritmalariga fikr-mulohaza beradi va foydalanuvchi e’tibor bermagan muammolarni aniqlaydi.
3. Foydalanuvchilar SDK orqali aniq va bilvosita signallarni log qilishlari mumkin bo‘ladi.
4. Foydalanuvchilar GPT-3 uchun yaratilgan yaxshilash mexanizmi yordamida modellarda va promptlarda A/B testlarni osonlik bilan o‘tkazishlari mumkin bo‘ladi.
5. Foydalanuvchilar turli modellarda promptlarni solishtirishlari, eng yaxshi modelni topishlari va xarajatni kamaytirishlari mumkin bo‘ladi.
