# Xulosa

Ushbu sahifa men ko‘proq IDElarga kirish imkoniga ega bo‘lganim sari yangilanadi.

Ushbu bob sizga qiziqarli bo‘lishi mumkin bo‘lgan ba’zi vositalarga umumiy nazar tashladi
prompt muhandisi sifatida. Quyida qaysi vositani ishlatishni va
qanday holatda foydalanishni tavsiya qilaman. E’tiborda tutingki, prompt muhandisligi tez rivojlanayotgan soha bo‘lib,
ilgari tilga olingan ko‘plab vositalar kelajakda sezilarli o‘zgarishlarga uchraydi.

## Yagona Prompt Iteratsiyasi uchun

Everyprompt yagona prompt iteratsiyasi uchun eng yaxshi funksiyalar to‘plamiga ega ko‘rinadi (men sinab ko‘rgan IDElar orasida). Oddiy playground ham yaxshi va biroz sodda.

## Prompt Chaining uchun

Dust hozirda prompt chaining uchun eng yaxshi (kamroq texnik) vosita hisoblanadi. U juda
kuchli funksiyalar to‘plamini taqdim etadi.

## Embeds

Dyno yagona embed taklif qiladigan vositadir.

## To‘liq Nazorat uchun

Langchain eng yaxshi tanlov! Bu Python kutubxonasi bo‘lib, to‘liq kengaytiriladi,
va allaqachon ajoyib funksiyalar to‘plamiga ega.

## Boshqalar

Ko‘proq IDElarga kirish imkoniga ega bo‘lganim sari yana tavsiyalar qo‘shaman. Men
turli vositalarni sinab ko‘rishni tavsiya qilaman, chunki har birining o‘ziga xos uslubi va turli funksiyalari bor.
