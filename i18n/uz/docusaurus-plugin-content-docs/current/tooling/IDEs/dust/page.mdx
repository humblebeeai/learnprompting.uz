## Xususiyatlar

Dust bir nechta birlashtiriladigan "bloklar" ko'rinishida kuchli vositalarni taqdim etadi,
masalan, LLM so'rovlari, kod parchalarini ishlatish va internetda qidiruv funksiyalari. Dust shuningdek,
ma'lumotlar to'plamlaridan foydalanish va promptlarni avtomatik tarzda ma'lumotlar to'plamlariga nisbatan sinab ko'rishni qo'llab-quvvatlaydi.

Hozirgi Dust funksionalligi bitta prompt ustida takrorlashdan ko'ra, promptlarni zanjirlashga qaratilgan.

Dust bir nechta model provayderlarini qo'llab-quvvatlaydi: (OpenAI, Cohere), va kelajakda HuggingFace hamda Replicate uchun qo'llab-quvvatlash rejalashtirilgan. Barcha provayderlar uchun API kalitlari talab qilinadi.

Dust'da qurilgan LLM ilovalarini joylashtirishingiz mumkin.
