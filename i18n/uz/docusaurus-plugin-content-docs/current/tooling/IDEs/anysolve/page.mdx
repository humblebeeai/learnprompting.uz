# AnySolve

[AnySolve](https://www.anysolve.ai) bu asosan Text promptlariga asoslangan vazifalarni yaratish va ulashish uchun mo‘ljallangan veb-sayt.



## Xususiyatlar

Vazifalarni UI muharririda yaratish mumkin. Yaratilgan vazifalar boshqa foydalanuvchilar bilan ulashilishi mumkin, shunda hamma vazifalarga kirish imkoniga ega bo‘ladi. REST API yordamida vazifani boshqa dasturiy ta’minotga integratsiya qilish mumkin.
