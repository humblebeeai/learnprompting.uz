# Soaked

[Soaked](https://soaked-prompts.vercel.app) bu prompt muhandisligi uchun mo'ljallangan IDE bo'lib,
grafik interfeys orqali vizual prompt zanjirlarini yaratishga yo'naltirilgan. Hozirda
beta bosqichida, biroq faol rivojlantirish to'xtatilgan.



## Xususiyatlari

Soaked promptlarni bir-biriga zanjir qilib ulash uchun grafik interfeysni taqdim etadi,
unda turli komponentlarning qanday bog'lanishini oson ko'rishingiz mumkin. Shuningdek,
bir vaqtning o'zida bir nechta natijalarni yaratish imkoniyati bilan past darajadagi prompt ko'rinishini ham taqdim etadi.
Shuningdek, natijalar uchun taxminiy xarajatni ko'rsatadi. Hozirda faqat GPT3 ni qo'llab-quvvatlaydi va foydalanish uchun OpenAI kaliti talab qilinadi.
