# Umumiy ko'rinish

Prompt engineering iterativ dizayn vazifasidir. Ko'proq rasmiylashtirilgan iterativ dizayn
sohalarida (masalan, logo dizayni) dizaynerlarga
samaraliroq ishlash imkonini beradigan yaxshi yo'lga qo'yilgan vositalar mavjud. Prompt engineering sohasida ham shunga o'xshash vositalar mavjud,
yaqinda turli prompt engineering IDElari (integratsiyalashgan dasturlash muhitlari)
yaratilgan. Bunday vositalarning ajablanarli darajada keng doirasi mavjud,
ilmiy-tadqiqot interfeyslaridan tortib, professional tarzda ishlab chiqilgan IDElargacha.
Ushbu bobda sizga prompt engineer sifatida qiziqarli bo'lishi mumkin bo'lgan
ba'zi vositalar haqida umumiy ma'lumot beriladi.

Ushbu bobning yakuniy maqolasida foydalanish uchun tavsiya qilinadigan vositalarim bilan tanishing.

Navbatda turishlar sababli, bu yerda tilga olingan barcha IDElarga hali kira olmadim.
Bundan tashqari, vositalar soni ko'payib borayotgani sababli, hammasining alohida sahifasi yo'q.

Quyida ushbu bobda muhokama qilinadigan vositalar jadvali keltirilgan:
