# Everyprompt

[Everyprompt](https://www.everyprompt.com) bu keng imkoniyatlarga ega bo'lgan prompt muhandisligi uchun IDE hisoblanadi
va yagona prompt iteratsiyasi uchun professional interfeysni taqdim etadi.



## Xususiyatlar

Everyprompt turli GPT modellari, ma'lumotlar to'plamlari,
va statistikani o'z ichiga olgan turli xususiyatlarga ega. Ushbu xususiyatlarning ba'zilaridan foydalanish uchun ishlab chiquvchilarga email yozish talab qilinishini unutmang,
shuning uchun ular qanchalik to'liq joriy etilgani hozircha aniq emas. Shuningdek, ular **50 mingta bepul** token ham taqdim etadi.
