# PromptTools

[PromptTools](https://github.com/hegelai/prompttools/) ochiq manbali kutubxona hisoblanadi
promptlar, LLMlar va vektor ma'lumotlar bazalari bilan tajriba o'tkazish, sinash va baholash uchun.

## Xususiyatlar

PromptTools quyidagi uchta rejimda promptlar va modellarning taqqoslanishi uchun maydoncha taqdim etadi:

- Ko'rsatma
- Prompt shabloni
- Modelni taqqoslash

Maydonchadan tashqari, PromptTools test yozish uchun SDK ham taqdim etadi
va promptlarni keng ko'lamda tajriba qilish va baholash uchun baholash funksiyalari.

Siz qo'llanma videosini [bu yerda](https://www.youtube.com/watch?v=cLGRqNI-nJU) ko'rishingiz mumkin.
