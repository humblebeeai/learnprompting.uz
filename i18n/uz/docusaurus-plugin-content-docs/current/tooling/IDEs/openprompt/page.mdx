# OpenPrompt

[OpenPrompt](https://thunlp.github.io/OpenPrompt/index.html) bu Python kutubxonasi
turli promptlarni yaratish va sinash uchun funksiyalarni taqdim etadi.



## Xususiyatlari

OpenPrompt huggingface modellari bilan ishlaydi, lekin hali OpenAI ni qo‘llab-quvvatlamaydi. U ko‘proq
ma’lumotlar to‘plamlari va testlarga e’tibor qaratadi, chaining emas. OpenPrompt, shuningdek,
soft-prompting ni https://github.com/thunlp/OpenDelta orqali qo‘llab-quvvatlaydi.
