# Lang Chain

[LangChain](https://github.com/hwchase17/langchain/) bu Python kutubxonasi
turli promptlarni yaratish va zanjirlash uchun turli funksiyalarni taqdim etadi.



## Xususiyatlari

LangChain provayderlarni oson almashtirish, LLMlar bilan o‘zaro aloqalarni standartlashtirish imkonini beradi,
va ramziy mulohaza va MRKL arxitekturasi kabi narsalar uchun _agentlar_ taqdim etadi (@karpas2022mrkl).

LangChain kelajakda grafik foydalanuvchi interfeysini qo‘shishni rejalashtirmoqda.
