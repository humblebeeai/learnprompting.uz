# Mantiq yurituvchi va harakat qiluvchi LLMlar

ReAct (@yao2022react)(reason, act) bu til modellari uchun murakkab muammolarni hal qilish imkonini beruvchi paradigma hisoblanadi
tabiiy til asosida fikrlashni talab qiladigan vazifalar. ReAct LLM mo'ljallangan vazifalar uchun yaratilgan bo'lib, unda
muayyan harakatlarni bajarishga ruxsat berilgan. Masalan, MRKL tizimidagi kabi, LLM bajarishi mumkin
tashqi API lar bilan o'zaro aloqada bo'lib, ma'lumotlarni olish uchun foydalanadi. Savol berilganda, LLM
ma'lumotni olish uchun biror harakatni bajarishni tanlashi va so'ngra savolga javob berishi mumkin
qaytarib olingan ma'lumotlarga asoslanib.

**ReAct tizimlarini MRKL tizimlari sifatida tasavvur qilish mumkin, ular qo'shimcha ravishda **mantiqiy fikrlash** qobiliyatiga ega
ular bajarishi mumkin bo'lgan harakatlar haqida**

Quyidagi rasmni ko'rib chiqing. Yuqoridagi katakdagi savol HotPotQA (@yang2018hotpotqa) manbasidan olingan,
murakkab mulohazani talab qiladigan savol-javoblar ma'lumotlar to'plami. ReAct savolga javob bera oladi
avvalo savol haqida mulohaza yuritish (O'y 1), so'ngra harakatni bajarish (Harakat 1) orqali so'rov yuborish
Google'ga. So'ngra u kuzatuv (Obs 1) ni oladi va bu fikr, harakat, kuzatuv bilan davom etadi
xulosa (3-qism) ga yetguncha davom etadi.
